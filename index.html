Projects

Here are some of my projects:

UCLA Gym Traffic (2016)

Using entrance data to UCLA facilities, I helped students choose the right time and place to go work out. This was my proudest project and it involved: writing a proposal to request data, simulating traffic patterns using statistical models, highlighting the "present" in graphics, and writing responsive code on the frontend.

Visualizing Stats Program (2016)

I made this visualization to help Statistics students pick classes and plan for their major. Working on this project improved my design intuition and ability to visualize relations.

Easy Classes (2016)

I analyzed the grade distribution data and made interactive tools to help students pick GE classes.

Data Visualization Workshop (2016)

I ran a workshop for Stats Club members, introducing them to data visualization.

Kaggle Crime Contest (2015)

In this machine learning contest, the biggest challenge was to estimate a probability distribution for each observation, rather than simply provide a one-shot classification. To deal with this challenge, I tried random sampling and feature engineering. Having done other Kaggle competitions, I learn the most from this.

Air Sensor Dashboard (2015)

I built a dashboard to provide an overview of CO2 level, temperature and humidity. It also allowed users to examine the relationship between any two variables and interactively lookup the raw data.

UCLA DataFest (2015)

Three teammates and I participated in UCLA's annual data hackathon where we analyzed data provided by a car sales website. I picked up a lot of new tools and this event kickstarted my passion for writing code to solve data problems.

IBM Watson Case Presentation (2014)

I worked with two teammates to envison deploying the IBM Watson AI in Chase bank branches.

Honduras Microfinance Trip (2014)

I was part of a group of ten students who traveled to a Honduran village to learn from and advise a local microfinance co-op.

Bathrooms of UCLA (2015)

Inspired by a blog post on a fun type of graphics called Chernoff Faces, I rated bathrooms at UCLA and used facial features to represent variables like cleanliness, privacy, etc.
